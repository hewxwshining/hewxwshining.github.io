flex

聊一聊 flex: 0 1 auto 的问题

flex 是 flex-grow、flex-shrink、flex-basis 这三个属性的缩写, 即

flex: <flex-grow> <flex-shrink> <flex-basis>;

  • flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

  • flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

  • flex-basis属性定义了在分配多余空间之前，项目占据的主轴空间（main size）， flex-basis 值会覆盖掉 width 值。

    item{
    flex: 0 1 auto;
}
或
.item {
    flex-grow: 0; // 增长比例，子项合计宽度小于容器宽度，需要根据每个子项设置的此属性比例对剩下的长度进行分配
    flex-shrink: 1; // 回缩比例，子项合计宽度大于容器宽度，需要根据每个子项设置的此属性比例对多出的长度进行分配
    flex-basis: auto; // 设置了宽度跟宽度走，没设置宽度跟内容实际宽度走
}

当主轴空间宽度有剩余， 子元素的最终的宽度是如何得到的？

例如：

<style type="text/css">
    .parent {
        display: flex;
        width: 600px;
    }

    .parent>div {
        height: 100px;
    }

    .item-1 {
        width: 140px;
        flex: 2 1 0%;
        background: blue;
    }

    .item-2 {
        width: 100px;
        flex: 2 1 auto;
        background: darkblue;
    }

    .item-3 {
        flex: 1 1 200px;
        background: lightblue;
    }
</style>
<div class="parent">
    <div class="item-1"></div>
    <div class="item-2"></div>
    <div class="item-3"></div>
</div>

  1. 项目容器整体宽度是 600px;

  2. 初始未伸缩的真实宽度为 0 + 100 + 200 = 300px

    • item-1 因为 flex-basis 设置了 0% ，所以宽度为 0px;
    • item-2 因为 flex-basis 设置了 auto ，同时设置了 width: 100px ，所以宽度随 withd 走，为100px;
    • item-3 因为 flex-basis 设置了 200px ，所以宽度为 200px;

  3. 那么还剩余 600-300 = 300 的宽度可以放大；

  4. 放大计算的方式为；

    • 所有子项目放大比列为 2:2:1 ；

    • 那么各个元素最终宽带分别为原来的宽带加上放大的宽带 ；

      item-1: 0 + 300 * 2/5 = 120px
      item-2: 100px + 300 * 2/5 = 220px
      item-3: 200px + 300 * 1/5 = 260px

当主轴空间宽度不足， 子元素的最终的宽度是如何得到的？

<style type="text/css">
    * {
        padding: 0;
        margin: 0;
    }

    .parent {
        width: 600px;
        height: 300px;
        display: flex;
    }

    .item-1 {
        /* 285 */
        flex: 1 2 500px;
        background: red;
    }

    .item-2 {
        /* 314 */
        flex: 2 1 400px;
        background: blue;
    }
</style>

<div class="parent">
    <div class="item-1"></div>
    <div class="item-2"></div>
</div>
  1. 项目容器整体宽度是 600px;
  2. 初始未伸缩的真实宽度为 500px + 400px = 900px， 需要缩放的宽度为 900px - 600px = 300px;
  3. 总权重 500 * 2 + 400 * 1 = 1400;
  4. 需要缩小的宽度：
    item-1: (500 * 2 / 1400) * 300 = 214.285714286
    item-2: (400 * 1 / 1400) * 300 = 85.714285714
  5. 最后的宽度：
    item-1: 500 - 214.285714286 = 285.714285714px
    item-2: 400 - 85.714285714 = 314.285714286px
