SVG实现基本动画

SVG动画

实现动画的方式有很多种，在这里主要是使用CSS3动画三剑客来实现（animation、transform、transition），
在说SVG前先介绍一下所以的CSS3动画三剑客。

1.animation

animation是如下几个属性的缩写，可以将动画与元素绑定，与keyframe共同使用。

语法

animation: name duration timing-function delay iteration-count direction fill-mode play-state;
  • animation-name：规定需要绑定到选择器的 keyframe 名称；
  • animation-duration : value：规定完成动画所花费的时间，以秒或毫秒计；
  • animation-timing-funtion ： value(linear|ease|ease-in|ease-out|ease-in-out|timing-funtion：规定动画的速度曲线；
  • animation-delay : value： 规定在动画开始之前的延迟；
  • animation-iteration-count : value|infinite：规定动画应该播放的次数；
  • animation-direction : normal|reverse|alternate|alternate-reverse：规定是否应该轮流反向播放动画；
  • animation-fill-mode ： none|forwards|backwarks|both:设置动画在执行之前和之后如何将样式应用于其目标;
  • animation-play-state: paused|running：设置动画是否运行和停止;

2.tranform

transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。

语法

transform: none|transform-functions;
  • rotate：旋转
  • translate：移动
  • skew：倾斜
  • scale：缩放

3.transition

transition是如下几个属性的缩写，可以实现样式到样式的过度效果，与animation有些相似，都可以做些简单的动画效果，但是transition没法想animation一样做到帧动画。

语法 

transition: property duration timing-function delay;
