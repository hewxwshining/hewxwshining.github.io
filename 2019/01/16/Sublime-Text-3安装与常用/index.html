Sublime Text 3安装与常用插件

sublimetext是前端的一个神器，以其精简和可diy而让广大fans疯狂。

在win10系统下安装sublime Text ,每次都会折腾一番，总会遇到这或那的问题，记录一下流程，跟遇到的问题以及解决的方法。

一、安装sublime

在官网上下载win10下的sublime版本，直接安装，这没什么好说的。

二、安装插件

Sublime 有很多插件，这些插件为我们写代码提供了非常强大的功能，这些插件需要单独安装。
而安装这些插件最方便的方法就是通过Package Control的插件，这其实就是一个插件管理器，帮助我们从Sublime中安装，升级和移除插件，这样对于我们管理插件来说是非常方便的，安装好它之后在通过它来安装其他插件。

这里有两种方式安装package Control

  • Ctrl+Shift+p -> install package control ->回车
  • 在菜单栏tools -> install package control

安装成功的情况应该会有这样的提示：

但是一般来说总会失败：

如果失败了我们就要直接去下载package control

package Control 安装源码地址：https://packagecontrol.io/installation

下载后的文件名称为：Package Control.sublime-package，将其解压到Package Control文件夹(注意文件夹名称首字母必须大写)，然后将文件夹拷贝到C:\Users\Administrator\AppData\Roaming\Sublime Text 3\Packages (即 Preferences –> Browse Packages打开的路径)

三、常用插件
1.emmet

这个没有什么好说的，类似jQuery的语法，编码蹭蹭往上提。不过要求PyV8环境（安装完后你会看到有一个文件夹），最好还是选择在线装吧。

PyV8 下载地址：https://github.com/emmetio/pyv8-binaries，将下载的文件解压到
PyV8文件夹（文件夹名唯一不可更改），然后将文件夹拷贝到C:\Users\Administrator\AppData\Roaming\Sublime Text 3\Installed Packages中，（即 preferences –> Browse Package打开的路径的上一层路径下的Installed Packges中）

2. 侧边栏增强插件SideBarEnhancements
3. 控制台呼出插件Terminal
Tips:快捷键 ctrl+shift+T呼出当前文件路径的控制台
4. 代码提示插件SublimeCodeIntel
5. 代码排版插件Sublime-HTMLPrettify
Tips:安装完快捷键ctrl+shift+h 一键格式化代码
6. CSS3前缀补充插件Autoprefixer
Tips:使用方法：在输入CSS3属性后（冒号前）按Tab键
7. 一键浏览文件

Preferences->Key Bindings - User打开用户快捷键设置，如下设置

[
    {
        "keys":["f12"],//快捷键
        "command": "side_bar_files_open_with",
        "args": {
            "paths": [],
            "application": "C:\\Users\\v_wxiahe\\AppData\\Local\\Google\\Chrome\\Application\\chrome.exe",//浏览器地址
            "extensions": ".*"
        }
    }
]

Tips:查了下默认的快捷键，SublimeText3中f1-f12中只有f11被默认为全屏命令，其他的没设置。也就是说，你可以装十个八个浏览器一字排开按过去测试。
