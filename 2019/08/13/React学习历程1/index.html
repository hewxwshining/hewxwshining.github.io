什么是React?

React 是一个声明式，高效且灵活的用于构建用户界面的 JavaScript 库。使用 React 可以将一些简短、独立的代码片段组合成复杂的 UI 界面，这些代码片段被称作“组件”。

静态资源引用

build后，访问项目，静态资源（static） 路劲不对，原因是我们对 react 项目打包上线时，文件夹build并不是在根目录下，这样出现服务器访问 index 找不到static 下面的资源，在浏览器中打开发者，发现访问静态资源路径变成了根文件

在 package.json 中 添加homepage属性

重新打包后静态资源访问路径为相对路径
