解决 npm run build 卡住问题

问题描述：

在终端执行 npm run build 时卡住，没有任何错误提示：

解决方案：

（1） 注释掉 build/check-versions.js 中检测代码

（2）执行 npm config set registry http://registry.cnpmjs.org 修改为淘宝源 npm config set registry https://registry.npm.taobao.org

解决

注：亲历证明，问题解决了，但是打包的过程中还是会出现卡顿的情况，而且时间长，效果不好：

总结

（1）改变为淘宝镜像源后，不注释check-version.js中的代码也是可以的，但是验证版本的时间会超级长，难以忍受；
